# UTSWeb
# Toko Buku Pink

Website toko buku sederhana bertema pink dan putih, dibuat untuk UTS Pemrograman Web 1.
Project ini dibangun menggunakan HTML, CSS, dan JavaScript (tanpa framework).

# Fitur

Login dengan validasi email & password
Stok / Katalog Buku lengkap dengan gambar cover

Form pemesanan buku (checkout) dengan perhitungan total otomatis

Tracking pesanan berdasarkan nomor DO

Data disimpan sementara menggunakan LocalStorage
